GrowRig device pages are generated from `growrig-platform/devices/<category>/<slug>/device.yaml`.

The directory path supplies the stable category and device slug. The YAML definition supplies user-facing metadata such as brand, model, connection type, Home Assistant integration, and semantic capabilities.

<Aside type="tip" title="Add documentation beside the device">
  A device directory may also contain `guide.md`. Its content is appended to the generated reference page, keeping setup notes, limitations, and troubleshooting beside the machine-readable definition.
</Aside>

Continue to the [device catalog](./catalog/) to browse all current definitions.
